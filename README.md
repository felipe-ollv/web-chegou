# ChegouApp

Painel para gestão de recebimentos em condomínios, baseado no template **Material Dashboard React** e limpo para customização.

## Como rodar

- `npm install`
- `npm start`

## O que já foi limpo

- Menus de exemplo trocados por uma única rota `Início`.
- Botão/flutuante de configurações e call-to-action de upgrade removidos.
- Página inicial simplificada em `src/layouts/home/index.js` para servir como ponto de partida.

## Onde editar

- Rotas: `src/routes.js`
- Layout base: `src/layouts/home/index.js`
- Temas e cores: `src/assets/theme` e `src/assets/theme-dark`
- Componentes compartilhados: `src/components`
